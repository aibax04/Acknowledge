# 🗑️ Delete Functionality Added

You can now delete **Notifications** and **Tasks**.

## ✅ Features Added

1. **Delete Notifications**:
   - **Who**: Seniors and Managers (who created them).
   - **Where**:
     - **Senior Dashboard**: In the "Sent Notifications" section.
     - **Manager Dashboard**: In the Notification Tray (top right bell icon).
   - **How**: Look for the **Trash Icon** or **Delete Button** next to the notification.

2. **Delete Tasks**:
   - **Who**: Managers (and task creators/assignees).
   - **Where**: **Manager Dashboard** > **Team & Tasks** tab.
   - **How**: A **Trash Icon** (Delete button) is available in the "Actions" column for each task.

## 🧪 How to Test

### 1. Delete a Notification
1. Log in as **Senior** or **Manager**.
2. Send a test notification (e.g., "Test Delete").
3. Find it in your list (Senior: Dashboard bottom right; Manager: Top right bell).
4. Click the **Trash/Delete Icon**.
5. Confirm the action.
6. The notification will be removed immediately.

### 2. Delete a Task
1. Log in as **Manager**.
2. Go to **Team & Tasks**.
3. Create a test task.
4. Find it in the task list.
5. Click the **Trash Icon** (red delete button) in the Actions column.
6. Confirm the action.
7. The task will be removed.

## 🔒 Permissions
- **Notifications**: Only the creator or a Senior can delete.
- **Tasks**: Only the creator, the assignee, or a Senior/Manager can delete.
